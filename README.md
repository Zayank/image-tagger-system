## Environment

Xampp Mysql Module & php Artisan CLI
PHP 7.3.28 & Mysql Ver 15.1 Distrib 10.4.19-MariaDB

## Steps for integration
- composer update
- set env parameters
- php artisan migrate
- php artisan storage:link
- run postman collection to ensure all tests are successful
